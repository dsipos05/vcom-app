Client Statistics Legend

  • DEC:
    Disconnect Event Count
  • DSCD:
    Disconnect State Cumulative Duration
  • SARAE:
    Send Audio Rate After Encoding (Kbps)
  • SAPLLS:
    Send Audio Packet Loss Last Second (%)
  • SAPLLM:
    Send Audio Packet Loss Last Minute (%)
  • SAPLSL:
    Send Audio Packet Loss Since Login (%)
  • SA Jitter:
    Send Audio Packet Jitter (ms)*
  • RARBD:
    Receive Audio Rate Before Decoding (Kbps)
  • RAPLLS:
    Receive Audio Packet Loss Last Second (%)
  • RAPLLM:
    Receive Audio Packet Loss Last Minute (%)
  • RAPLSL:
    Receive Audio Packet Loss Since Login (%)
  • RA Jitter:
    Receive Audio Packet Jitter(ms)*
* The first jitter value is the highest network jitter recorded in the last second and the second jitter value is the current jitter buffer size.
