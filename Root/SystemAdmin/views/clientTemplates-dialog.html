Templates

  • You must select two or more clients to link them
  • If none of the selected clients have a template of the chosen type*, then use the first client selected as the base template
  • If exactly one selected client has a template of the chosen type*, then use that template as the base template
  • If multiple clients have the same template of the chosen type*, then use that template as the base template
  • If multiple clients have different templates of the chosen type*, then no linking can happen.

    • *: There are three template types (Selector Assignments, Audio Settings, and Options)

 
 
 
 

Hold Ctrl to select additional items, or hold Shift to select several in a row
