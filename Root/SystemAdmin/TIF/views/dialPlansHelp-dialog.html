Dial Plan Help

Syntax

  • Type
    Value
    Result
  • Digit
    0 1 2 3 4 5 6 7 8 or 9
    Identifies a specific digit
    (do not use #)
  • Wild Card
    x
    Matches any single digit that is dialed
  • Wild Card
    . (period)
    Matches an arbitrary number of digits
  • Timer
    T
    Indicates that an additional timeout period of 4 seconds should take place before automatic dialing starts

Examples

  • Dial Type
    Digits Dialed
    Dial Plan
  • Internal Extension
    A four digit number
    xxxx
  • Emergency
    911
    911
  • Local Operators
    9 (if required for an outside line) then 0
    90T
  • Local Number
    9 (if required for an outside line) and then a seven digit number
    9xxxxxxxT
  • Long Distance
    9 (if required), 1, area code and local number
    91xxxxxxxxxx
  • International
    9 (if required), 011, then any number of additional digits (at least one)
    9011x.T
